<h1> Calculator v1 </h1>
A simple command-line calculator built with Python.
This program performs basic arithmetic operations between two integers based on user input.

<h2> Overview </h2><br>
This project was created to practice fundamental Python concepts, including: <br>
-User input handling <br>
-Conditional statements <br>
-Basic arithmetic operations <br>
-Program execution via terminal <br>

<h2> Features </h2><br>
-Addition (+) <br>
-Subtraction (-) <br>
-Multiplication (*) <br>
-Division (/) <br>

<h2>Requirements</h2><br>
Python 3.x installed

<h2>How to Run</h2><br>
1. Open the terminal in the project directory. <br>
2. Run: <br>
python calculator_v1.py <br>
3. Enter: <br>
-First number <br>
-Second number <br>
-operation symbol (+, -, *, /) <br>

<br>

<h2>Example</h2><br>
Enter a number: 8 <br>
Enter another number: 4 <br>
The calculation is +, -, *, or /: / <br>

<h2>Output:</h2><br>
2.0

<h2>Limitations</h2><br>
-Accepts only integers <br>
-No validation for invalid inputs <br>
-Does not prevent division by zero <br>
-Uses separate if statements instead of elif <br>
